# NetPi-Scanner
CS576 final project

## Required features
- discovery of devices
- performance for devices
- saving reports to csv

## UI
- start off with simple options such as scan, performance, and then view csv results

## Insallation Instructions

Clone the repository
```
git clone https://github.com/RomelAleman/NetPi-Scanner.git
```

Remove the .git file
```
cd path/to/NetPi-Scanner && rm .git
```

Run initilization script
```
sudo ./init.py
```
